# 加密货币长线做空策略

> 基于 Binance 永续合约数据的量化做空策略研究，聚焦因子有效性验证与稳定收益实现

## 📊 研究结论速览

### 核心发现

| 因子 | IC | IC_IR | 6年方向一致性 | 结论 |
|------|-----|-------|-------------|------|
| **vol_14d_z** (截面波动率) | -0.078 | **-0.287** | ✅ 全6年一致 | **最佳做空因子**，做空高波动币 |
| **neg_mom_z** (负动量) | +0.052 | +0.212 | ✅ 6年一致 | 做空追高币，IC方向为正 = 做空高动量币赚钱 |
| **fr_vol_adj_z** (费率/波动率比值) | +0.050 | +0.232 | ✅ 6年一致 | 做空低费率/低波动比值币 |

### ⚠️ 关键警示

- **因子有效 ≠ 策略赚钱**：vol_z IC-IR 优秀（-0.287），但2023-2025牛市期间做空策略实际亏损
- **择时是核心**：只有在熊市/高波动市场，做空策略才能提供正收益
- **资金费率因子不稳定**：2021年为负（做空高费率正确），2022年后翻转（做多高费率才对）

## 🔬 数据概况

- **来源**：Binance 永续合约 (USDT本位)
- **时间跨度**：2021-01 ~ 2026-03
- **币种数量**：634 个币种（K线 + 资金费率 + 链上数据三数据源交集）
- **分析粒度**：日频截面数据
- **数据字段**：收盘价、成交量、资金费率、taker买卖流量、链上流通量

## 📈 主要分析结果

### 1. IC/IR 总体统计

```
因子          IC_mean   IC_IR    IC>0占比
vol_14d_z    -0.078    -0.287    38.2%   ← 做空高波动
vol_7d_z     -0.074    -0.277    38.9%   ← 做空高波动
mom_14d_z    -0.053    -0.203    41.9%   ← 做空高动量
skew_z       -0.040    -0.166    42.9%   ← 做空右偏分布
fr_z         +0.016    +0.066    53.3%   ← 做多高费率(不稳定)
```

### 3. 2×2 分组收益（做空组每日收益）

| 组合 | 日均收益 | 年化收益 | Sharpe | 胜率 |
|------|---------|---------|--------|------|
| HighVol × HighFR | +0.386% | **+110%** | 1.22 | 51.7% |
| HighVol × LowFR | +0.034% | -39.2% | 0.11 | 50.1% |
| LowVol × HighFR | +0.101% | -10.2% | 0.38 | 52.7% |
| LowVol × LowFR | +0.104% | -6.5% | 0.41 | 53.1% |

> 注意：HighVol×HighFR 年化+110% 是**做空该组合的收益**，即做空高波动+高费率币种有显著正收益

## 🧪 研究方法

1. **因子构建**：截面标准化（z-score），去除极端值
2. **IC计算**：Spearman秩相关系数，每日截面计算
3. **IR验证**：IC均值/标准差，衡量因子稳定性
4. **分组回测**：Top20% vs Bottom20%，双档验证
5. **择时分析**：全市场资金费率阈值过滤
