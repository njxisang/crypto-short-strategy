## 研究结论汇总

### 核心结论（2026-04）

**做空策略的有效性取决于市场环境**：

1. **vol_z（截面波动率）** 是最强做空因子，IC_IR = -0.287，6年稳定
2. **纯做空策略在牛市中必然亏损**——因子有效不代表正收益
3. **择时是关键**：只在熊市/高波动环境中启动做空仓位
4. **资金费率因子不可独立使用**：需要配合择时
